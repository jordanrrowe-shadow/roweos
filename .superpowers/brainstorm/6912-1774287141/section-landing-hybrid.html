Social Hub Landing

Same hybrid pattern -- marketing header, live stats, 4 primary features as cards, secondary features as compact pills. Consistent across all complex sections.

The Hybrid Pattern:

  • Top: Marketing-style header with section tagline (Georgia serif, uppercase label)
  • Stats bar: Live metrics pulled from real data (not static)
  • Primary features: 2x2 card grid with icon + title + description (clickable)
  • Secondary features: Compact pill row for less-used tabs
  • Click any card/pill → enters that sub-section with breadcrumb + tab bar
  • Consistent template across Automations, Social, Mail, Studio, System
