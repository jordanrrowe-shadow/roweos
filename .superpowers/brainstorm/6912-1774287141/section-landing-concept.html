A: Feature Card Grid

Marketing-site style cards in a grid. Each card has an icon, title, and description. Click a card to enter that sub-section. Stats bar at the bottom for context. Elegant, spacious, explains each feature.

B: Navigable List

Clean list with icons, titles, descriptions, and chevrons. More compact, feels like navigating into subsections of an app. Less marketing, more functional. Still explains each feature before you enter.

How it works (both options):

  • Click a sidebar nav item (e.g., "Social") → see the landing page first
  • Click a feature card/row → enter that sub-section with breadcrumb + tab bar
  • Breadcrumb shows: Home › Social › Engage
  • "Back" button or clicking section name in breadcrumb returns to the landing page
  • Tab bar still available inside for switching between sub-sections without going back
  • Onboarding choice: "Simple" vs "Advanced" controls whether landing pages show by default
